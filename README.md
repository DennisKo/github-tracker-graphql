# Github Tracker

List and edit issues for Github repositories using the Github GraphQL API.

---

### Setup

```bash
yarn
```

Edit `apollo.config.js` and `src/index.tsx` and add your `GH_TOKEN`.

### Develop

```bash
yarn start
yarn codegen:watch
```

### Build

```bash
yarn codegen
yarn build
```

(Optional)
Run it

```bash
npx serve dist
```
